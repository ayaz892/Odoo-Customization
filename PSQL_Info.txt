TO CONNECT WITH YOUR DB ON LOCALHOST USING TERMINAL
___________________________________
1) Open terminal and write this command
sudo -u postgres psql

2) Add this command with your DB Name
psql -U odoo -d your_db_name

3) Write this command with your DB name again
\c your_db_name

4) It should show this message "You are now connected to database "your_db_name should be show here" as user "postgres".

5) Now you can use sql commands to do operations like fetching adding values into table

 IF DB On local is expire how to extend its expiry date and reactive it.
______________________________________________________________________

run these command one by one on terminal
1) sudo -u postgres psql
2) \c your_db_name
3) UPDATE ir_config_parameter 
SET value = '2099-12-31' 
WHERE key = 'database.expiration_date';
4) \q (for exit from psql) 
After it re login on db

How to change the password for DB Backup on localhost
____________________________________________________
run these command one by one
1) 1) sudo -u postgres psql
2) \c your_db_name
3) update res_users set password='password you want to set' where login='email of the user here';
